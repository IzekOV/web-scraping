# Web Scraping

This project scrapes book data from a website and saves it to CSV.

## Features
- Extract book title
- Extract price
- Extract rating
- Extract categories

## Language & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas

## How to run

git clone https://github.com/IzekOV/web-scraping

cd web-scraping

pip install -r requirements.txt

python scraper.py
